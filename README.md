# Customer Contact Manager

## Overview

The Customer Contact Manager is a desktop application built with Python, Tkinter, and SQLite. The application allows users to manage customer records by adding, updating, searching, and deleting customer information.

## Features

* Add customer records
* Store names, phone numbers, and email addresses
* Search customer information
* Update customer records
* Delete customer records
* Save data using SQLite

## Technologies Used

* Python
* Tkinter
* SQLite

## How to Run

1. Install Python 3.
2. Download the project files.
3. Open a terminal in the project directory.
4. Run:

python customer_contact_manager.py
